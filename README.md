## Description

Live example with GitHub Actions to build a funcional pipeline

## Installation

1 - Clone the repository

2 - Instal and run [Docker Desktop](https://www.docker.com/);


```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
